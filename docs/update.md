# Обновление модуля

Обновление модуля происходит так же, как и установка, через админку Opencart. Всё делается автоматически, единственное, что нужно будет сделать после установки - зайти в настройки модуля TeleCart, чтобы применились послеустановочные скрипты. Это обязательное требование после каждого обновления.
