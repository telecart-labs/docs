TeleCart поставляется с продвинутым и очень гибким функционалом по созданию форм, основанном на фреймворке [FormKit](https://formkit.com/). Он позволяет добавлять любое количество полей ввода, задавать валидацию, иконки, конфигурировать сложные динамические формы и многое другое.

Одно из применений этого функционала - форма оформления заказа, где администратору магазина доступна возможность использовать как заранее определённые поля, которые созраняются в стандартные поля заказа OpenCart, так и любое количество любых произвольных полей, которые будут сохранены в мета информацию OpenCart заказа. Поля ввода цвета, дат, дней недели, выпадающих меню и многих других доступных типов полей, которые можно добавить к заказу, что удовлетворит потребности любых бизнесов.

Доступно 2 режима работы:

1. визуальный конструктор. Позволяет создавать и настраивать простые формы просто перетягивая элементы на поле. ПОдходит для пользователей, не разбирающихся в разработке.
2. Редактор кода. Раскрывает функционал полностью, позволяя с помощью JSON синтаксиса использовать функционал фреймворка по полной, создавая динамику, условные операторы, кастомную верстку и т.д. TeleCart поддерживает почти весь функционал, описанный в документации https://formkit.com/essentials/schema.
