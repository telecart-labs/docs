# Трассировка

!!! note

    Страница находится в разработке и ещё не готова.

TeleCart имеет встроенные механизмы замера времени выполнения всех HTTP запросов. Для этого используется интеграция с Sentry.

## Включение Sentry

В `.env` прописать следующие переменные окружения:

```
SENTRY_ENABLED=false
SENTRY_DSN=
SENTRY_TRACES_SAMPLE_RATE=1.0
```
